# FontForge Web — Конвертер шрифтов в браузере

Одностраничное веб-приложение для конвертации шрифтов между форматами TTF, OTF, WOFF (и SVG при импорте) с акцентом на генерацию веб-шрифтов и CSS-кода `@font-face`. Работает полностью на стороне клиента, без отправки файлов на сервер.

![screenshot](https://github.com/thetimyr/FontForge-Web/blob/main/other/Frame%201%20(2))

## ✨ Возможности

- 📥 **Импорт шрифтов** из файлов (TTF, OTF, WOFF, WOFF2, SVG) через drag-and-drop или диалог выбора.
- 🖥️ **Загрузка системных шрифтов** (Chrome 103+, требует разрешения пользователя).
- 👁️ **Предпросмотр в реальном времени** с настраиваемым текстом и размером.
- ℹ️ **Детальная информация**: семейство, начертание, вес, количество глифов, версия, копирайт.
- 📤 **Экспорт в форматы**:
  - `WOFF` (рекомендован для веба, со сжатием таблиц)
  - `TTF` (TrueType)
  - `OTF` (OpenType)
- 🎨 **Автоматическая генерация CSS** `@font-face` с копированием в буфер обмена.
- ⌨️ **Горячие клавиши** (`Ctrl+O` — импорт, `Ctrl+S` — экспорт WOFF, `Ctrl+E` — экспорт TTF).
- 🌐 **Полностью офлайн** — все библиотеки загружаются через CDN, но после первого открытия кэшируются.

## 🚀 Быстрый старт

### Локальный запуск

1. Скачайте файл [`index.html`](index.html) из этого репозитория.
2. Откройте его в любом современном браузере (Chrome, Edge, Firefox, Opera).
3. Начните работу: перетащите шрифт в специальную область или нажмите «Системные шрифты» (требуется Chrome).

**Никаких серверов, сборщиков или зависимостей устанавливать не нужно.**

## 🛠️ Технические детали

### Используемые библиотеки

- [opentype.js](https://opentype.js.org/) (1.3.4) — чтение и запись шрифтов OpenType/TrueType.
- [pako](https://github.com/nodeca/pako) (2.1.0) — сжатие zlib для создания WOFF.

Все библиотеки подгружаются с CDN, поэтому интернет-соединение требуется только при первом запуске (либо для загрузки системных шрифтов через API).

### Поддерживаемые форматы

| Формат | Импорт | Экспорт | Примечание |
|--------|--------|---------|------------|
| TTF    | ✅     | ✅      | Полная поддержка |
| OTF    | ✅     | ✅      | Включая CFF |
| WOFF   | ✅     | ✅      | Ручная упаковка со сжатием |
| WOFF2  | ✅     | ❌      | Требуется декомпрессия Brotli; при конвертации используйте TTF/OTF |
| SVG    | ✅     | ❌      | Устаревший формат, импорт через парсинг SVG-контуров |

> Экспорт всегда создаёт валидные TTF/OTF/WOFF. Исходный SVG-шрифт конвертируется во внутреннее представление и может быть экспортирован в любой поддерживаемый формат.

### Системные шрифты

Для работы с локальными шрифтами используется [`window.queryLocalFonts()`](https://developer.mozilla.org/en-US/docs/Web/API/Window/queryLocalFonts) (доступен в Chrome 103+, Edge 103+). Браузер запросит разрешение на чтение списка установленных шрифтов. После выбора шрифта он загружается в память и становится доступным для предпросмотра и конвертации.

## 📖 Инструкция по использованию

1. **Выберите источник шрифта**  
- *Системные шрифты*: нажмите кнопку «🖥️ Системные шрифты», дайте разрешение, затем кликните по нужному шрифту в списке.  
- *Файлы*: перетащите один или несколько файлов в область «📁 Перетащите файл сюда» или нажмите на неё, чтобы открыть проводник.

2. **Предпросмотр**  
В центральной панели вы увидите выбранный шрифт. Измените текст и размер по своему усмотрению.

3. **Экспорт**  
В правой панели нажмите на нужный формат:  
- **WOFF** — для использования на веб-сайтах (рекомендовано).  
- **TTF** или **OTF** — для десктопных приложений.  
Файл будет загружен автоматически.

4. **CSS-код**  
При экспорте в WOFF автоматически обновляется блок с CSS. Нажмите «📋 Копировать», чтобы скопировать правило `@font-face` в буфер обмена. Можно также нажать «🔄 Сгенерировать CSS» в любой момент.

5. **Сброс**  
Кнопка «🗑️ Сброс» очищает выбранный шрифт и список импортированных файлов.

## ⌨️ Горячие клавиши

| Сочетание | Действие |
|-----------|----------|
| `Ctrl+O`  | Импорт файла(ов) |
| `Ctrl+S`  | Экспорт в WOFF |
| `Ctrl+E`  | Экспорт в TTF |

## ⚠️ Известные ограничения

- **WOFF2**: импорт работает, но конвертация *из* WOFF2 в другие форматы может быть нестабильной, потому что opentype.js не поддерживает Brotli-декомпрессию. Рекомендуется предварительно конвертировать WOFF2 в TTF с помощью других инструментов.
- **SVG-шрифты**: импорт возможен только для простых шрифтов без сложных трансформаций.
- **Системные шрифты**: поддерживаются только в браузерах на Chromium. В Firefox и Safari функция недоступна.
- **Большие шрифты**: приложение может замедлиться на шрифтах с десятками тысяч глифов из-за объёма вычислений в браузере.
